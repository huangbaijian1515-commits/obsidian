# Prompt: Compile Sources

Use this prompt with Codex from the vault root.

## Task

Scan `00_Inbox/` and `10_Sources/` for new or unreviewed source material. Produce a human-reviewable extraction draft. Do not promote anything into `20_Query/` without explicit approval.

## Output

For each source:

- source note path
- source type
- status
- quality estimate
- missing metadata
- extracted viewpoints
- extracted judgments
- extracted facts
- extracted data
- suggested query notes
- suggested topic links
- possible duplicates
- privacy concerns

## Rules

- Preserve uncertainty.
- Do not invent missing facts.
- Keep every extracted unit traceable to its source.
- Mark sensitive or work material clearly.
- Prefer fewer, higher-quality query candidates.
