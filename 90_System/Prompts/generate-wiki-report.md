# Prompt: Generate Wiki Report

Use this prompt with a topic name.

## Task

Read related notes in `20_Query/`, `30_Topics/`, and relevant `10_Sources/`. Generate a wiki report in `40_Wiki/` using `90_System/Templates/wiki-report.md`.

## Required Sections

- Executive Summary
- Knowledge Map
- Core Claims
- Evidence
- Tensions / Contradictions
- Open Questions
- Next Reading / Next Capture

## Rules

- Separate evidence from interpretation.
- Include Mermaid graph syntax for the knowledge map.
- Identify conflicting claims explicitly.
- Prefer reviewed query notes over draft query notes.
- Mention important missing evidence.
