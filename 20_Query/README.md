# Query

Reviewed atomic knowledge lives here. These notes are the compounding layer of the vault.

Only promote content into this folder after human review.

Use one of four note types:

- `claim`: a factual or interpretive statement that can be supported, contested, or superseded
- `concept`: a reusable idea, model, mechanism, or definition
- `case`: an event, example, personal record, meeting, interview, or observed pattern
- `data`: a number, metric, table conclusion, or quantitative observation

Use `90_System/Templates/query-note.md`.
